# Fonts for FourierDrawer (opentype.js)

Place **Roboto-Regular.ttf** here so the Fourier epicycle text works without relying on external CDNs.

- Download: [Google Fonts – Roboto](https://fonts.google.com/specimen/Roboto) → download family → use `Roboto-Regular.ttf` from the static folder.
- Or from [google/fonts](https://github.com/google/fonts/blob/main/apache/roboto/static/Roboto-Regular.ttf) (raw).

The app will use `/fonts/Roboto-Regular.ttf` first; if that fails, it tries CDN URLs and then falls back to a circle animation.

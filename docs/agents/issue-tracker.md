# Issue Tracker

**Platform:** GitHub Issues
**Repository:** https://github.com/vietbui1999ru/vietbui1999ru.github.io/issues

## Agent instructions

- Create issues via `gh issue create`
- Reference issues by number: `#123`
- Link PRs with `Closes #123` in PR body
- Before starting features: `gh issue list --label ready-for-agent`

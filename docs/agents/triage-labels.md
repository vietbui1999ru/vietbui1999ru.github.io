# Triage Labels

| Label | Meaning |
|-------|---------|
| `needs-triage`    | Maintainer needs to evaluate |
| `needs-info`      | Waiting on more info from reporter |
| `ready-for-agent` | Fully specified, AFK-ready |
| `ready-for-human` | Needs human judgment |
| `wontfix`         | Will not be actioned |

## Agent instructions

- Pick up work filtered by `ready-for-agent`
- New issues default to `needs-triage`
- Under-specified → apply `needs-info`, comment what's missing
